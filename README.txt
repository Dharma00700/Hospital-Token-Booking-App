
# Hospital Token Booking App

Tech: Python (Flask), HTML, CSS, JavaScript

## How to Run (Windows/Mac/Linux)
1) Open a terminal in this folder.
2) Create venv (optional) and install deps:
   python -m pip install --upgrade pip
   pip install -r requirements.txt
3) Start the app:
   python app.py
4) Open your browser at: http://127.0.0.1:5000

## Age → Room Rules
- 1–20  → Room 1
- 21–40 → Room 2
- 41–60 → Room 3
- 61–80 → Room 4
- 81–100 → Room 5

The token shows the assigned room and doctor, and you can print it.
